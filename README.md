# 斑马百科 · 产品营销周报看板

一个现代化的产品营销周报HTML看板原型，适合品牌营销团队在企微移动端查看。

## 📱 功能特性

### 核心功能
- **周次切换**：支持左右箭头在不同周次间切换
- **OKR进度**：可视化进度条展示OKR完成情况
- **数据总览**：售卖数据、下载拉新、站内周数据、站外曝光的全面展示
- **趋势图表**：使用Chart.js展示近8周关键指标趋势
- **本周重点**：富文本编辑器（Quill.js），支持多种格式
- **下周计划**：可编辑表格，支持新增/删除行

### 权限系统
- **查看者模式**（默认）：所有内容只读
- **管理者模式**：输入密码后解锁编辑功能
  - 密码：`zebra2026`
  - 可编辑：OKR、本周重点、下周计划
  - 可操作：生成快照

### 快照系统
- 管理者可生成当前数据快照
- 快照列表在右侧边栏查看
- 点击可回溯历史版本（只读模式）
- 快照数据保存在浏览器 localStorage

### 智能隐藏
- 站外曝光数据模块在无数据时自动隐藏
- 示例：W21 的站外数据为空，切换到该周时模块不显示

## 🚀 部署指南

### 方式一：直接打开
双击 `index.html` 即可在浏览器中运行（需联网加载CDN资源）。

### 方式二：本地HTTP服务
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .

# PHP
php -S localhost:8080
```

然后访问 `http://localhost:8080`

### 方式三：部署到服务器
将整个 `weekly-dashboard/` 目录上传至任意静态文件服务器（Nginx、Apache、Vercel、Netlify等）。

### 方式四：企业微信H5
1. 将文件部署到HTTPS域名下
2. 在企微管理后台配置可信域名
3. 通过链接分享到群组即可

## ⚙️ 自定义指南

### 修改密码
编辑 `js/app.js` 中的 `ADMIN_PASSWORD` 常量：
```javascript
const ADMIN_PASSWORD = "your_new_password";
```

### 添加新的周次数据
在 `js/data.js` 的 `MOCK_DATA` 对象中添加新的周Key（如 `W24`），并更新 `AVAILABLE_WEEKS` 数组。

### 修改图表样式
编辑 `js/charts.js` 中的 `chartDefaults` 配置对象。
